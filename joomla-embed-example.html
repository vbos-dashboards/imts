Pacific Islands Trade Dashboard

This is an example of how to embed the Trade Dashboard in your Joomla website. The dashboard displays comprehensive trade data including NSDP indicators, trade balances, exports, imports, and dietary risk factors.

How to Use This in Joomla

Copy and paste the following code into your Joomla article or custom HTML module:

<div class="responsive-iframe-container">
  <iframe
    src="https://yjulio.github.io/trade_dashboard"
    title="Trade Dashboard"
    allow="fullscreen">
  </iframe>
</div>

<style>
.responsive-iframe-container {
  position: relative;
  width: 100%;
  padding-bottom: 75%;
  height: 0;
  overflow: hidden;
  border-radius: 8px;
}

.responsive-iframe-container iframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  border: 0;
}
</style>

Note: The dashboard will be available at https://yjulio.github.io/trade_dashboard after deployment.
